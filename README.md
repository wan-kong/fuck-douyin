# 抖音净化 fuck-douyin

一个用于精简抖音网页版的浏览器脚本，让刷视频和看直播更清爽。

它可以自动跳过广告、直播和购物内容，播放时优先选择最高画质，还能隐藏页面上不需要的按钮和直播特效。所有功能都可以单独开关，设置会自动保存。

## 它能做什么

### 刷视频时

- 自动跳过广告、直播卡片和购物内容
- 自动切换到最高画质
- 按需隐藏顶部栏、左侧导航栏
- 按需隐藏头像、点赞、评论、收藏、分享、“听抖音”和更多按钮

### 看直播时

- 隐藏礼物横幅和全屏礼物动画
- 屏蔽彩色、特效和付费弹幕
- 隐藏直播间顶部标签及推荐入口
- 自动开启“屏蔽送礼信息”
- 自动切换到最高画质

## 安装

### 在线安装

1. 安装浏览器扩展 [Tampermonkey](https://www.tampermonkey.net/)、Greasemonkey 或 Violentmonkey。
2. 打开 [Greasy Fork 发布页](https://greasyfork.org/zh-CN/scripts/583463-%E6%8A%96%E9%9F%B3%E5%87%80%E5%8C%96-douyin)，点击“安装此脚本”。
3. 在脚本管理器中确认安装。
4. 打开[抖音网页版](https://www.douyin.com/)，页面右下角出现悬浮按钮即表示脚本已启用。

### 从源码安装

下载本项目，在项目目录运行：

```bash
pnpm install
pnpm build
```

打开 `dist/` 目录，将生成的 `.user.js` 文件拖入浏览器并确认安装。

## 使用方法

点击页面右下角的悬浮按钮即可打开设置面板。按钮可以拖到屏幕四角，脚本会记住它的位置。

设置分为“视频”和“直播”两组，打开或关闭选项后会立即生效，无需刷新页面。下次打开抖音时，之前的设置仍会保留。

## 默认设置

安装后默认启用以下功能：

- 跳过广告和直播卡片
- 视频与直播自动选择最高画质
- 隐藏直播礼物横幅、礼物动画和高级弹幕
- 隐藏直播间标签
- 屏蔽送礼信息

自动跳过购物内容、隐藏页面区域和播放器侧边按钮默认关闭，可在面板中按需开启。

## 常见问题

### 安装后没有看到悬浮按钮

确认脚本管理扩展和“抖音净化”脚本均已启用，然后刷新 `www.douyin.com` 或 `live.douyin.com` 页面。

### 某个功能突然失效

抖音网页结构更新后，部分隐藏或自动操作功能可能暂时失效。可以提交 Issue，并附上失效页面、功能名称和浏览器版本。

### 脚本会收集数据吗

不会。脚本只在浏览器本地修改页面显示和交互，配置也保存在本地，不收集或上传个人数据。

## 开发

项目使用 pnpm：

```bash
pnpm dev      # 启动本地开发环境
pnpm build    # 类型检查并生成用户脚本
pnpm preview  # 预览构建结果
```

构建产物位于 `dist/`。项目没有单独的 lint 或 test 命令，`pnpm build` 是提交前的主要检查。
